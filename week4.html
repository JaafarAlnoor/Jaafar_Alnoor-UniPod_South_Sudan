Week 4: CAD Implementation - CorelDRAW Technical Drawings

Date: October 14 - October 20, 2025

Quick Summary

  • Skills Learned: Vector drawing, precision dimensioning, layer management, technical documentation, file export formats
  • Digital Tools and Software Used: CorelDRAW Graphics Suite, PDF export tools, DXF conversion utilities
  • Topics: Vector graphics for technical drawing, precision techniques, engineering standards, manufacturing documentation
  • Key Concepts: Vector vs raster graphics, coordinate-based precision, technical drawing standards, scalable design
  • Challenges Faced: Software learning curve, precision requirements, file compatibility, technical standard adherence, workflow optimization

Introduction

This week marked a significant transition from conceptual design to technical implementation. Building upon the Design Thinking process from Week 3, I focused on creating precise technical drawings of the Secure Shoe Rack System using CorelDRAW, a professional vector graphics software.

The objective was to translate the human-centered design concept into manufacturing-ready technical documentation that clearly communicates dimensions, assembly methods, and construction details for the lock-and-key shoe storage system.

Tool Configuration:
• Dimension Tool: Added to main toolbar

• Object Manager: Enabled for layer control
• Color Palette: Technical colors (black, red, blue)
