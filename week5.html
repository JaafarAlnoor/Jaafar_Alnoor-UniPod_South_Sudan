Week 5: 3D Part Design - SolidWorks Component Modeling

Date: October 21 - October 27, 2025

Quick Summary

  • Skills Learned: 3D parametric part modeling, sketch constraints, feature creation, material application, design intent
  • Digital Tools and Software Used: SolidWorks Part Environment, Sketch Tools, Feature Manager, Material Editor
  • Topics: Parametric part design, feature-based modeling, design for manufacturing, technical documentation
  • Key Concepts: Design intent, parametric relationships, feature hierarchy, material properties, manufacturing considerations
  • Challenges Faced: Complex feature creation, design intent preservation, parametric relationships, file management

Introduction

This week focused exclusively on 3D part design using SolidWorks, translating the 2D technical drawings from Week 4 into parametric 3D components. The emphasis was on creating individual parts with proper design intent, manufacturing considerations, and accurate material properties.

The objective was to master SolidWorks part modeling techniques to create manufacturable components for the Secure Shoe Rack System, with each part designed to precise specifications and ready for production.

Process Documentation

The 3D part design process involved creating individual components for the Secure Shoe Rack System using SolidWorks parametric modeling techniques, translating the 2D technical drawings from Week 4 into manufacturable 3D parts.

Main Cabinet Frame (4 Steps)

1Base Sketch & Grid Division Creation

Figure 1: Front Plane selection and base sketch creation

Sketch Creation Process:
• Selected Front Plane as sketch plane for base geometry
• Created 334mm × 211mm rectangle using Center Rectangle tool
• Applied geometric relations: Horizontal, Vertical, Equal
• Fully defined sketch with all dimensions (black lines)
• Added construction lines for reference geometry
• Established origin at center for symmetric modeling
2Frame Extrusion

Figure 2 & 3: Extrusion parameters and feature configuration

Extrusion Process:
• Applied Extruded Boss/Base feature to create 3D geometry
• Configured extrusion parameters for precise depth control
• Used equation-driven dimensions for parametric design
• Verified feature creation with error checking enabled
3Back Cover Creation

Figure 4 & 5: Back Cover

Grid Creation Process:
• Created new sketch on the extruded front face
• Used Linear Sketch Pattern for 8 vertical dividers
• Used Linear Sketch Pattern for 5 horizontal shelves
• Divider thickness: 6mm (material specification)
• Compartment internal size: ~30.5mm × 31.4mm
• Applied Extruded Cut feature through all (45mm)
• Used the existing solid frame as reference geometry

Door Unit Construction (2 Steps)

4Door Panel Creation

Figure 5: Door panel sketch and extrusion

Door Panel Design:
• New part file created for door component
• Outer dimensions: 38mm × 38mm square
• Frame thickness: 6mm (matches main frame material)
• Inner panel: 35mm × 35mm clear area
• Extrusion depth: 6mm (single material thickness)
• Clearance: 0.5mm allowance for smooth operation
5Final Door Assembly

Figure 5 & 6: Completed door with all manufacturing features

Final Features and Verification:
• Applied 0.5mm fillets to all sharp edges
• Added 1mm chamfers to external corners
• Verified mass properties: ~25g per door
• Checked center of mass for balance
• Performed interference detection with frame
• Added custom properties for manufacturing

Design Intent Preservation: All components maintain parametric relationships with the original 2D technical drawings. Dimensions are driven by equations and global variables to ensure consistency across all parts. Manufacturing considerations include proper fillets, chamfers, and clearance allowances for assembly.

Reflection

Learning Outcomes

This week provided deep immersion in 3D part design and parametric modeling principles:
  • Parametric Modeling Expertise: Mastered feature-based modeling techniques, understanding how to create intelligent, easily modifiable parts that maintain design intent through changes.
  • Design for Manufacturing: Developed awareness of manufacturing considerations while designing parts, including draft angles, fillets, and material selection.
  • Sketch Mastery: Gained proficiency in creating fully defined sketches with proper geometric relations, forming the foundation for robust 3D features.
  • Technical Documentation: Understood the direct relationship between 3D part models and 2D manufacturing drawings.

Challenges and Solutions

  • Design Intent Preservation: Initially struggled with features breaking during modifications, but learned to use relations and equations for robust parametric design.
  • Complex Feature Creation: Faced challenges with advanced features but mastered the use of reference geometry for precise feature placement.
  • File Organization: Implemented systematic naming conventions and folder structures for multiple part files.

Future Applications

The skills developed this week in parametric 3D modeling will be directly applicable to future projects requiring precise component design and manufacturing documentation. The understanding of design intent and feature-based modeling provides a solid foundation for more complex assemblies and advanced CAD workflows.
